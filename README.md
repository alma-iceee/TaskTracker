## Prerequisites

[Docker Desktop](https://www.docker.com/products/docker-desktop/)

## Installation

Download from git using this command:

```
git clone https://github.com/alma-iceee/TaskTracker.git
```

Run docker-compose:

```
docker-compose up
```
